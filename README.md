## how to use

To clone and run this app you will need:

```bash
# clone a repository
$ git clone https://github.com/ericdev0212/spring-react-charts.git

# Enther project folder
$ cd spring-react-charts

# Install the dependencies 
$ yarn

# run the application
$ yarn dev
```
